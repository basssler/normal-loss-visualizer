What Is the Normal Loss Function?

The normal loss function does not measure the probability of a stockout. It measures the expected size of the stockout beyond the reorder point, in standardized units.

If demand ends at the reorder point or below, shortage is zero. If demand lands past the reorder point, shortage equals demand minus the reorder point. The normal loss function summarizes the average size of that overshoot in standardized units.

Left of R: no stockout, so shortage = 0.
Just right of R: there is a stockout, but the shortage is small.
Far right of R: there is a stockout, and the shortage is much larger.
Key idea: the red shaded tail is a probability. The normal loss function is not that shaded area. It measures average overshoot beyond the reorder point in standardized units.

Show the math

Show the math trail
Step 1: Convert the reorder point into a z-score
z = (R - mu_X) / sigma_X
Step 2: Use z to find the normal loss function
NL(z) = f(z) - z[1 - F(z)]
Step 3: Convert standardized overshoot into actual units short
n_R = sigma_X * NL(z)
Standard normal PDF: f(z) = (1 / sqrt(2pi)) * e^(-z^2 / 2)
Standard normal CDF: F(z) = integral from -inf to z of f(t) dt

Cost Section

Cost model used here: proportional order cost = cD, setup cost = K(D / Q), holding cost = h(Q / 2 + safety stock), shortage cost = p(D / Q)n_R.
